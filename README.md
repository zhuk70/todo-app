# ToDo App 🚀

Aplicação web desenvolvida com React para gerenciamento de tarefas.

## 📌 Funcionalidades
- Criar tarefas
- Editar tarefas
- Marcar como concluída
- Deletar tarefas
- Persistência com localStorage

## 🛠 Tecnologias
- React
- JavaScript
- CSS

## ▶️ Como rodar o projeto

```bash
npm install
npm run dev
